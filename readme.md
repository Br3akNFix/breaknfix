# BreaknFix

**BreaknFix** es mi diario técnico donde documento mi ruta hacia la certificación **CCNA** y mis primeros pasos en **ciberseguridad**.  
El objetivo es aprender rompiendo y arreglando: cada error es una oportunidad de entender más a fondo las redes y la seguridad informática.

## Objetivos
- Documentar cada subtema del CCNA con el método Cornell.
- Crear un glosario breve y un índice modular de conceptos.
- Explorar fundamentos de ciberseguridad aplicados a redes.
- Compartir diagramas, comandos y casos prácticos.

## Estructura
- **Glosario** → definiciones rápidas estilo libro.
- **Conceptos** → explicaciones profundas con ejemplos y comandos.
- **Subtemas** → índice modular de cada capítulo del CCNA.
- **Ciberseguridad** → notas paralelas sobre firewalls, protocolos seguros y ataques comunes.
